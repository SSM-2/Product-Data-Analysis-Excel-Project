# Product Performance Dashboard (Excel BI)

An interactive, 360-degree retail analytics solution designed to visualize product performance, customer demographics, and geographical trends from **2015 to 2023**.

---

## Project Overview
This dashboard transforms raw, fragmented sales data into a centralized command center. It allows stakeholders to monitor high-level KPIs and perform deep-dive analysis using dynamic multi-dimensional filters.

## Key Performance Indicators (KPIs)
The dashboard tracks five primary metrics to provide an immediate snapshot of business health:

| Metric | Value |
| :--- | :--- |
| **Total Revenue** | $798.1k |
| **Total Orders** | 10,308 |
| **Popular Product** | Grains |
| **Average Revenue** | $77.42 / transaction |
| **Top Performing State** | Texas |

## Visualizations & Insights
The report utilizes a variety of specialized charts to tell the data story:

* **Sales Trend (Line Chart):** Visualizes the steady revenue growth trajectory.
* **Best Selling Products (Bar Chart):** Compares volume across specific items (Bread, Pasta, Eggs).
* **Category & Brand Analysis:** Horizontal and vertical charts detailing performance across categories (Grains, Dairy, Meats, Fruits) and Brands Aâ€“E.
* **Gender Insights (Donut Chart):** A 52/48 split analysis of the customer base.
* **Customer Concentration (Map):** A geographic heat map highlighting U.S. sales density.

---

## Interactivity
Located on the left sidebar, the **Slicers** allow for real-time data drilling:
- [x] **State:** Filter the entire dashboard by specific US regions.
- [x] **Years:** Drill down into specific performance years.
- [x] **Category:** Focus on segments like Dairy, Meats, or Grains.

---

## Technical Stack
* **Microsoft Excel:** Core platform.
* **Power Query:** Used for the ETL (Extract, Transform, Load) process to ensure data integrity.
* **PivotTables & PivotCharts:** For complex data aggregation.
* **Slicers:** Implemented for seamless interactive report connections.
